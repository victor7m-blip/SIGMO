# SIGMO v0.2

Base profissional do SIGMO em React + Vite + Supabase.

## Rodar localmente

```bash
npm install
npm run dev
```

Login inicial de teste, se o `supabase.sql` já foi executado:

- RE: `admin`
- PIN: `123456`

## Publicar

Depois de enviar para o GitHub, a Vercel publica automaticamente.

## Observação

Esta versão ainda usa login simples por RE/PIN para protótipo operacional. A autenticação definitiva será endurecida em fases posteriores.
